# 지하철 노선도 미션
스프링 과정 실습을 위한 지하철 노선도 애플리케이션


# 기능 요구사항 

- [ ] 출발역과 도착역 사이의 최단경로를 구할 수 있다.
  - [ ] 출발역에서 갈 수 있는 모든 구간을 구할 수 있다.
  - [ ] 갈 수 있는 모든 구간 안에 도착지가 있는지 확인할 수 있다.
- [ ] 해당 경로의 거리와 요금을 계산한다.
  - [x] 최단 경로의 거리를 구할 수 있다.
  - [ ] 이동 거리마다 추가 운임이 부과된다.
